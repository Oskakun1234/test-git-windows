■ アプリ概要 タスクの記録・管理をWeb上で決着するシンプルなタスク管理アプリです。 日常生活や業務において「すぐにメモしたい」「タスクを整理したい」というニーズを想定して開発しました。

このアプリは、これまで学習してきたJavaの知識を踏まえ、設計から実装まで一貫して1人で開発を完遂することを目的として制作しています。

■ 利用技術 バックエンド：Java（Servlet / JSP） フロントエンド：HTML / CSS / JavaScript データベース：MySQL サーバー：Apache Tomcat 開発環境：Eclipse IDE

■ 主な機能 ユーザー登録 / ログイン機能（認証機能） タスクの投稿機能 タスクの編集 / 削除機能

■工夫した点 セキュリティ対策 SQLインジェクション対策としてPreparedStatementを使用 不正なリクエストへの基本的なバリデーションを実装 UI/UXの改善 シンプルで直感的に操作できる画面構成を意識 最低限の操作でタスク管理ができる設計 エラーメッセージをわかりやすく表示

■ 課題・今後の改善AC 現状は基本的な CRUD 機能の実装に残っているとりあえずは以下の機能を追加予定： タスクごとの期限設定 詳細情報（説明・優先度など）の並び替え・検索機能の追加
