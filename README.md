■ アプリ概要
タスクの記録・管理をWeb上で行えるシンプルなタスク管理アプリです。
日常生活や業務において「すぐにメモしたい」「タスクを整理したい」というニーズを想定して開発しました。

本アプリは、これまで学習してきたJavaの知識を活かし、設計から実装まで一貫して1人で開発を完遂することを目的として制作しています。

■ 使用技術
バックエンド：Java（Servlet / JSP）
フロントエンド：HTML / CSS / JavaScript
データベース：MySQL
サーバー：Apache Tomcat
開発環境：Eclipse IDE

■ 主な機能
ユーザー登録 / ログイン機能（認証機能）
タスクの投稿機能
タスクの編集 / 削除機能

■ 工夫した点
セキュリティ対策
SQLインジェクション対策としてPreparedStatementを使用
不正なリクエストへの基本的なバリデーションを実装
UI/UXの改善
シンプルで直感的に操作できる画面構成を意識
最低限の操作でタスク管理ができる設計
エラーメッセージを分かりやすく表示
